  1. 前往”自定义与控制Google Chrome”选单中选择”更多工具”->”扩展程序”。

  2. 拖曳Chome-DNA.crx file 到扩展程序页面并放开所拖曳的crx file。

  3. 点选在确认窗口上的按钮 “添加应用”。

  4. DNA Chrome Edition开始安装中…

  5. DNA Chrome Edition安装完毕。
